# Roadmap for Web Creators
> 

Roadmap.sh is a tool for creating mind maps, study plans, and paths for developers. Users will find maps focused on specific roles, listing the skills needed for that role, as well as skill-based maps that suggest what to learn for a particular skill.

You can find, for example, a [guide for UX designers](https://roadmap.sh/ux-design), a guide on [how to understand user research](https://roadmap.sh/r/jornada-de-junior-ux), or a [step-by-step process for creating a design system](https://roadmap.sh/design-system). It also serves developers, product people, and other roles. Besides the official roadmaps, the site also offers maps created by the community or generated with AI.

The project started as a set of visual graphs on GitHub and has now grown to include interactive features, such as clicking on learning resources, the ability to contribute to new maps, suggest changes, and track your own progress.
