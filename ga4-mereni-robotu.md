# Tracking robots on the web: PHP script for Google Analytics 4
> 

Marek Lecián prepared a PHP script for [measuring robots on the web in Google Analytics 4](https://mareklecian.cz/ga4-mereni-robotu-crawleru-aneb-pomocnik-pro-seo/). It tracks how search engines crawl your website. It allows you to analyze content indexing, uncover crawl traps, monitor attack attempts, and verify the functionality of rules in robots.txt.

The script is intended for users with basic PHP knowledge who can export the outputs to Google BigQuery for detailed analysis. The script does not slow down the website and is effective for measuring pages with HTTP status codes 200 and 404. A simple solution that facilitates website indexing management.
