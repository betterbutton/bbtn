# New Performance panel in Chrome DevTools
> 

The Chrome DevTools team has introduced an experimental version of a new [Performance panel](https://developer.chrome.com/blog/new-in-devtools-128#perf), which brings improved tools for optimizing web performance. The new panel provides deeper insights into Core Web Vitals, including both lab and field data for LCP, CLS, and INP, and offers new tools for simulating real-world conditions.

The redesigned panel features interactive elements and analytical tools that help developers more effectively identify and resolve web performance issues. These new features are available in Chrome Canary, and the team is encouraging the community to test them and provide feedback.

[A more detailed look at what's new in the Performance panel](https://www.debugbear.com/blog/fix-web-performance-devtools) is offered by Umar Hansa on Debug Bear.
