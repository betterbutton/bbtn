# CRAP
> CRAP (Contrast, Repetition, Alignment, Proximity) are fundamental principles of visual design that ensure clarity, consistency, and user-friendliness.

**CRAP** (Contrast, Repetition, Alignment, Proximity) was defined by [Robin Williams](https://en.wikipedia.org/wiki/Robin_Williams_(writer)) in the book *[The Non-Designer’s Design Book](https://www.goodreads.com/book/show/833034.The_Non_Designer_s_Design_Book)* in 1994. She wanted to help people without formal graphic education understand the basic principles of good design. CRAP is a practical guide that allows creating visually effective and professional designs. The principles are widely used not only in graphic design but also in user interface design.

**Contrast** (Contrast) highlights important elements using different colors, sizes, or shapes. It helps users easily distinguish individual parts and focus on essential information. For example, in a user interface, it is important that the *"Buy"* button is sufficiently distinct from other elements on the page; otherwise, users may easily overlook it.

**Repetition** (Repetition) is achieved by using the same visual elements, such as colors, fonts, and styles. The design becomes unified. Repetition promotes consistency and helps users navigate better. An example is a website where headings, buttons, and navigation elements always have the same appearance, making them easier to recognize.

**Alignment** (Alignment) ensures visual connection between elements and helps create a structured and professional look. Without it, design can appear messy and chaotic. For example, texts on a website should be aligned to one edge to be more readable and appear organized.

**Proximity** (Proximity) groups related elements together and separates unrelated ones. This improves content organization and makes it easier to understand. A typical example is a form – the *"Submit"* button should be right below the text fields to make it clear that it is related to them.

When designing interfaces, these principles are often violated: **Insufficient contrast:** Light text on a light background is poorly readable. The solution is to check [color contrast accessibility](https://webaim.org/resources/contrastchecker/). **Too many different fonts or colors:** The design appears discordant and distracting. It is recommended to use at most [two to three fonts](https://fontjoy.com/) and a limited [color palette](https://goodpalette.io/). **Poor alignment:** Randomly placed elements disrupt readability. It is important to adhere to a clear grid. **Insufficient use of proximity:** For example, if a label in a form is not directly above the input field, it may confuse users about where to enter the information.

In addition to the CRAP principles, you will also encounter other rules of visual design. [Gestalt principles](https://en.wikipedia.org/wiki/Gestalt_psychology) describe how the human brain naturally organizes visual information. [Principles of visual hierarchy](https://www.interaction-design.org/literature/topics/visual-hierarchy) then determine which elements attract attention first – for example, larger font sizes and more saturated colors attract the eye before small and inconspicuous elements.
