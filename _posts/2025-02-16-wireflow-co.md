# With Wireflow.co, design application or website flows
> Wireflow.co is a free online tool that allows you to create prototypes of user flows for websites or applications.

![Wireflow.co screenshot](https://substack-post-media.s3.amazonaws.com/public/images/ff0c8ba9-98c9-45b3-b5fa-d9909cab1b0c_1894x1079.png "1456x829")

Wireflow.co is a tool for rough design of screen sequences

[Wireflow.co](https://wireflow.co/) is a free online open-source tool with which you can easily create prototypes of user flows. It has an intuitive drag-and-drop interface that makes visualizing user journeys easy without requiring advanced design skills.

The application offers a library of more than a hundred customizable graphical and functional elements. This allows you to model even complex interactions between users and the interface. If you are wary of possible information leakage when using online tools, you will be pleased that Wireflow.co has its [source code on GitHub](https://github.com/vanila-io/wireflow), so you can run it on your own server or locally in Docker.
