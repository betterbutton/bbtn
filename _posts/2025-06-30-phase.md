# Phase is a tool for designing animated prototypes
> A no-code tool that allows UI/UX designers to easily create animated prototypes and focus on both creativity and usability.

![Screenshot of the prototype editing interface in the Phase tool](https://substack-post-media.s3.amazonaws.com/public/images/d97ee64c-53b1-4edc-9531-3ef393d50476_1265x671.png "1265x671")

[Phase](https://www.phase.com/) is a tool for designing animated prototypes

[Phase](https://www.phase.com/) is a tool for designing prototypes without coding. It helps designers quickly and simply create animated prototypes. No coding or complex animation knowledge is required. This makes it easy to design visually striking and interactive prototypes.

The main goal of Phase is to remove the barriers between an idea and its animated realization. The interface is simple and the features are powerful. Designers can thus focus on the design itself and the user experience.

Compared to common animation tools, Phase is designed specifically for user interfaces and can work with imports from Figma. It offers workflows that simplify even complex animation tasks. You can export the output as code.

Phase emphasizes intuitive work and fair sharing of responsibility. It's not just a tool, but also a community of designers who want to create without fear and without limitations.
