# Markdown in Google Docs
> 

For some time now, Google Workspace Docs has supported importing and exporting documents in Markdown format. A new feature now allows you to enable Markdown recognition while typing. This new capability works not only in Docs, but also in Slides and Drawings. If you want to write documents in Markdown, go to the Tools menu >
