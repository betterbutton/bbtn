## From The Designer

![Left: menu without highlighting the current page (wrong) and right: menu with the current page highlighted (correct).](https://substack-post-media.s3.amazonaws.com/public/images/2ae1020f-3101-445c-93cc-cf6869bd2e46_2160x1080.png "570x285")

**Show users where they are.** Website navigation must indicate the current location.\
\
❌ All items in the menu look the same\
✅ Current page highlighted in navigation\
\
👉 People understand the website structure more easily.\
👉 Frustration from searching is reduced.\
\
Clear navigation → a satisfied visitor.

## Linkodrome

**Shortcuts and gestures increase UI efficiency for advanced users.** Accelerators like keyboard shortcuts, gestures, or voice commands help experienced users [speed up routine tasks and increase work efficiency](https://www.nngroup.com/articles/ui-accelerators/) in the user interface. They serve as an alternative path to the same goal that beginners can ignore. The learning curve is supported by gradual disclosure without overwhelming novice users.

**How AI is changing the way we control software.** Chat-based interfaces, popularized by ChatGPT, once again recall the era of text terminals: powerful but not very user-friendly. [AI is reshaping traditional UX design patterns](https://allenpike.com/2025/post-chat-llm-ui): it enables context-aware right-click actions, natural language search (“find tax documents”), or intuitive selections (“next Tuesday” instead of clicking in a calendar), and of course [autocomplete](https://www.nngroup.com/articles/prompt-suggestions/). Inline feedback and automatic suggestions are also increasing. However, the real revolution could be completely generated UI – still with a question mark. Traditional software may soon seem outdated, thinks Allen Pike.
