# Magic link
> Magic link authentication is a passwordless login method. The user receives a one-time link via email and gains instant access to their account.

Magic link authentication ([magic link authentication](https://postmarkapp.com/blog/magic-links)) – or as Microsoft calls it: [magic connection](https://support.microsoft.com/cs-cz/topic/how-does-the-magic-link-option-work-5f0ca3f3-3f04-462a-b87d-781b705ff337) – is an interactive design pattern for a simple and secure way to log in without entering a password.

The user enters their email address and receives a link containing a one-time login token. Clicking the link verifies the user's identity. This method eliminates the need to remember passwords, reduces security risks associated with password leaks, and simplifies the login process.

One of the main advantages of magic links is their use in deferred registration, where users can start using a service without the need to set up a user account. Besides magic links, there are other passwordless authentication methods. However, magic links are one of the simplest ways to balance security and user convenience. This makes them an ideal solution.
