# Poline – color palette generator
> A color palette generator that connects colors with arcs in polar coordinates, bringing unexpected visual harmony.

![Screenshot from the Poline tool](https://substack-post-media.s3.amazonaws.com/public/images/30622552-0882-425e-b913-c79eb59f9a40_1904x1079.png "1456x825")

[Poline](https://meodai.github.io/poline/) generates color palettes.

[Poline](https://meodai.github.io/poline/) generates color palettes using arcs between anchor points in polar space. Instead of classic color interpolation, it offers a completely new, highly visual approach. It's suitable for creative coding, generative art, or even dynamic themes in user interfaces.

Unlike standard color mixing in RGB or HSL space, Poline uses polar coordinates. This creates smooth and often surprising transitions even between color-distant points. The color palette is formed using *anchor points*, specified in HSL or XYZ format, and their connection via so-called *position functions* – such as sine, quadratic, or exponential.

The developer has control over all parameters – number of points, loop closure, hue shift, or luminance inversion. Output colors can be obtained as HSL value arrays or directly as CSS strings. With the help of the external library [Culori](https://culorijs.org/), it's possible to convert colors to OKLCH or LCH models as well.

The interactive *playground* allows immediate experimentation with palettes. Click to add points, use arrows to shift hues or change the functions that define transition character. Poline is ideal wherever color needs motion, fluidity, and originality.

Install the library simply via `npm` or using a CDN. The project is open source and MIT licensed. If you're looking for another way to work with color – lively, intuitive, and surprisingly artistic – Poline will open new possibilities for you.
