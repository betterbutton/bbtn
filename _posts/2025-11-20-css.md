# Cascading Style Sheets
> A design language that separates the description of a website's appearance from its structure and content.

[Cascading Style Sheets](https://www.jakpsatweb.cz/css/), known as CSS ([cascading style sheets](https://www.w3.org/Style/CSS/Overview.en.html)), determine how content is displayed on the web – typography, colors, layout, and spacing – while the structure and meaning remain in HTML. They form the visual layer of the web and allow designers to control the appearance of the entire website from one place without repeating code on every page.

Cascading Style Sheets separate appearance from structure.

CSS was created in 1994 by *[Håkon Wium](https://www.wiumlie.no/enhttps://www.wiumlie.no/en)* [](https://www.wiumlie.no/enhttps://www.wiumlie.no/en) *[Lie](https://www.wiumlie.no/enhttps://www.wiumlie.no/en)* and in collaboration with *[Bert Bos](https://www.reknisioweb.cz/p/bert-bos)*, it was developed within the *W3C* consortium. It became an official standard in 1996. The cascading model determines which styles have precedence when multiple rules apply to the same element. It combines browser defaults, user choices, and author intent. Thanks to this, the web is adaptable and accessible – users can change fonts or contrast without interfering with the content.

Over time, CSS has evolved from a simple system of colors and fonts into a complex tool for interface design. CSS2 introduced positioning and support for various devices, CSS3 split the specification into modules, and today's CSS includes grid layouts, custom properties, variable fonts, and container queries. Collaboration among browser developers has ensured consistency and reliable behavior across platforms.

For designers and user experience creators, CSS represents much more than just page decoration. It determines visual hierarchy, readability, and how users perceive the structure of information. Its syntax is simple, but its impact is fundamental – thanks to CSS, the web has transformed from a text network into a full-fledged design medium that decides not only how the web looks but also how we feel about it.
