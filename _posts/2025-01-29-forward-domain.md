# ForwardDomain.net redirects your website
> Redirect your domain easily and for free with ForwardDomain.net! Define rules in DNS, get SSL and IPv6 support – no server or code needed.

![](https://substack-post-media.s3.amazonaws.com/public/images/ff1d7a84-8d60-4fee-a4e2-be7910f42fa7_1918x1079.png "1456x819")

[ForwardDomain.net](https://forwarddomain.net/) takes care of website redirection

Do you want to redirect a second-level domain to `www.` or move an entire domain to a new one? You will need an HTTP status code redirection for that. You can, of course, set up a web server and write a few lines of code for the redirection, or use the [ForwardDomain.net](https://forwarddomain.net/) service and define the entire redirection via a DNS record.

The service works by setting authoritative server records (`A` and `AAAA`) or alias (`CNAME`) records for your server, and then defining a redirection rule for the domain in a `TXT` record. ForwardDomain.net handles the SSL certificate and IPv6. You decide which HTTP status code to use. And it's all completely free.
