# Glyphter – a web utility for creating image-based fonts
> Glyphter is a freely available tool for creating fonts from SVG images. Without registration, it offers font export for the web and a style snippet for embedding.

![A screenshot from the Glyphter tool shows an area with a character table where vector images can be placed. On the right is a palette with ready-made vector icons that can be dragged into the table. At the top is a field for naming and downloading the font.](https://substack-post-media.s3.amazonaws.com/public/images/f52e1d61-22b8-4ffb-987e-27a50904ab02_1902x1055.png "1456x808")

[Glyphter.com](http://Glyphter.com) offers a simple interface for creating an icon font.

Sometimes it's practical to use a custom font for vector graphics on the web instead of individual SVG images. You know this approach from [Font Awesome](https://fontawesome.com/icons), which contains thousands of icons. Typical usage includes not only icons but also various logos and brands. In CSS, you can then handle the font using the [content property of pseudo-elements](https://developer.mozilla.org/en-US/docs/Web/CSS/::before). For icon fonts, it's good to remember accessibility. When read aloud, instead of the icon, the letter whose position in the font the icon occupies will be spoken. Use the [aria-hidden](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-hidden) or [aria-label](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-label) attribute.

If you're currently sighing that [font editors](https://alternativeto.net/software/glyphs-app/) are expensive, complicated, or only for macOS, you'll be pleased with our today's tip for a tool: [Glyphter](https://glyphter.com/) is a free web tool for building a font from SVG images. In its basic version, it requires no registration and is free. The output consists of fonts (.eot, .ttf, .woff, and .svg) and a style snippet for embedding fonts into a web page. If you miss the [OTF](https://en.wikipedia.org/wiki/OpenType) format, you can [convert the resulting font](https://transfonter.org/). The fonts will, of course, also work in programs like Word.

Glyphter is really just a basic tool and you won't use its more advanced features like kerning pairs, ligatures, or alternatives, nor do you have control over font metrics. For creating text typefaces, we would rather dissuade you from using it, but for icon fonts, it's an ideal tool for beginners.
