# Lazy loading
> Lazy loading is a technique that significantly speeds up web page loading. It loads only what the user actually needs – and only when they need it.

With [lazy loading](https://www.vzhurudolu.cz/prirucka/lazy-loading), parts of the page content, such as images, videos, or code snippets, are not loaded immediately when the page is opened. Instead, they are loaded only when they are truly necessary. This makes the page load faster, consume less data, and work more smoothly.

Modern browsers already have built-in support. You just need to add a simple attribute directly to the page code. But if you want more control, you can use various tools – for example, tracking when an element approaches the viewport, or asynchronous components that activate only when the user actually needs them.

Lazy loading isn't just for images and videos. It can also be applied to web analytics, ads, or other external scripts – in short, to anything that doesn't need to be loaded immediately.

Imagine a website as a library full of books. Loading everything at once is like borrowing all the books you might ever want to read to take home. Lazy loading, on the other hand, is like reaching for one specific book right when it's your turn. This makes the entire user experience simpler, faster, and more enjoyable.
