# Web fonts: how to set up and optimize
> 

Web fonts allow designers to control the visual appearance of text on pages. Using the `@font-face` declaration, various styles and weights of the font can be set. Variable fonts simplify this process thanks to flexible metrics for weight or slant. Pavel Satrapa[1](#footnote-1) wrote for Root [an article full of practical examples](https://www.root.cz/clanky/pisma-na-webu-webove-pismo-a-typograficke-speciality/).

For optimal performance, it is recommended to combine local and web fonts. The `font-display` property plays an important role in how the font is loaded and displayed. Typographic specialties such as ligatures or various numeral variants can enhance the design, but it always depends on the balance between aesthetic elements and loading efficiency.
