# Link directly to text on web pages
> 

Text fragments are an innovative feature of modern web browsers that [allows for precise linking to specific text](https://alfy.blog/2024/10/19/linking-directly-to-web-page-content.html#:~:text=Copy%20link%20to%20highlight) on web pages without the need for an anchor or ID. After adding special syntax to the URL, the browser will search for, highlight, and move the user directly to the spot in the document.

The feature is [fully supported in the Chrome browser](https://caniuse.com/?search=target-text) and can be styled using the pseudo-element [::target-text](https://developer.mozilla.org/en-US/docs/Web/CSS/::target-text). If you want to create a link simply, you just need to select the text, right-click, and choose "copy link to highlight" from the context menu. Some browsers do not allow easy link creation, but they can handle fragment links.

This technology simplifies navigation and brings new possibilities for content sharing, not only for technically savvy users but potentially for the general public through intuitive tools. One place where you will encounter text fragment links is, for example, Google's search results page.
