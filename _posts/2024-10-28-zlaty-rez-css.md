# The Golden Ratio in CSS
> 

[The Golden Ratio](https://cs.wikipedia.org/wiki/Zlat%C3%BD_%C5%99ez), also known as 1.618, has fascinated artists, architects, and designers for millennia. In nature and art, it creates harmony and is the basis for visually balanced compositions. From the pyramids to seashells, [this proportional principle can also be applied to web design](https://medium.com/design-bootcamp/the-golden-ratio-in-css-58995aa40903).

In CSS, it helps us create naturally [attractive and balanced layouts](https://tomlutzenberger.github.io/aspect-ratio.css/docs/). The Golden Ratio is not just a number, but a philosophy used by, for example, ancient architects and Renaissance artists. So why not use it in modern web design as well?

In Mads Stoumann's tutorial, you will learn [how to use CSS to create a diagram based on the Golden Ratio and the Fibonacci sequence](https://dev.to/madsstoumann/the-golden-ratio-in-css-53d0). Using a grid and a few tricks with pseudo-elements, you will be able to create a spiral effect. The code uses a grid with dimensions derived from the Fibonacci sequence, and both colors and positions of individual elements are set using custom properties.

Although CSS cannot directly draw spirals, we can mimic them using circles and their precise placement on a grid canvas.
