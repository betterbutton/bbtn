# Polzer: Search Console Data Is Enough For Me, I Don't Need Other SEO Tools
> 

Jan Polzer advocates using [Google Search Console](https://search.google.com/search-console/) as the primary tool for SEO analysis, instead of expensive and often underutilized paid SEO tools. [On his blog, he argues](https://www.maxiorel.cz/seo-nastroje-pouzivam-temer-vyhradne-jen-data-ze-search-console) that Search Console provides crucial data on rankings, traffic, and keyword performance without unnecessary monthly fees.

> "I am of the opinion that instead of loading data into specialized tools, selecting keywords to track, and constant monthly payments, you can perfectly manage with the free tool."\
> **— [Jan Polzer](https://www.maxiorel.cz/seo-nastroje-pouzivam-temer-vyhradne-jen-data-ze-search-console)**

Although the console doesn't answer everything, according to Honza it provides sufficient information to improve existing website content. Honza demonstrates how to use Search Console, including switching between metrics, comparing with previous periods, and highlights the benefits of custom reports in R Studio for more precise analysis.
