# Three Lessons from Art for Better UX Design
> 

[Sathish Chandler](https://www.linkedin.com/in/sathishchander/) selected three principles from architecture, art, and mythology that [can improve UX design](https://www.linkedin.com/pulse/how-3-unusual-tips-from-art-make-ux-better-sathish-chander/). The three key aspects are:

- **[Consistency](https://www.figma.com/resource-library/consistency-in-design/)** means creating harmony between different UI elements so they work as a whole. Consistency in UX is about similarity in action sequences in similar situations.

- **[Usability](https://cs.wikipedia.org/wiki/Pou%C5%BEitelnost)** focuses on clear communication that facilitates understanding of the interface for all types of users. Usability in UX is about clear communication of the relationship between the user and UI elements (buttons, menus, colors, contrast, sequence of steps, etc.).

- **[Locus of control](https://cs.wikipedia.org/wiki/Locus_of_control)** reminds that errors are inevitable and it's important to give users tools for correction, such as feedback and the ability to undo errors. The locus of control should be with the user... remember the history of actions and allow error correction.
