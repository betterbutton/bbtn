# AI in Data Analysis
> 

Marek Prokop in a post on Facebook describes [three main approaches to using artificial intelligence in data analysis](https://www.linkedin.com/feed/update/urn:li:activity:7272862980692656128/). The first is quickly inserting data into a chatbot, which helps with orientation in unfamiliar data, but has limitations in accuracy and size. The second way is connecting via API, ideal for large data and repeated tasks. The third option is programming with AI as a partner, where AI supports work in a programming language, which requires basic programming knowledge. Prokop emphasizes data security and welcomes more tips from readers.

> "This is my daily bread. Essentially, you tell the AI: 'Here are my data, this is how they look, this is what I need to get from them and use that programming language for it.'"\
> **— [Marek Prokop](https://www.linkedin.com/feed/update/urn:li:activity:7272862980692656128/)**
