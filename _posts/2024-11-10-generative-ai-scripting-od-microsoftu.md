# Generative AI Scripting from Microsoft
> 

[Microsoft introduced GenAIScript.](https://microsoft.github.io/genaiscript/) It is a tool for working with large language models (LLM) directly in JavaScript, which enables easy creation and sharing of prompts. It offers integration with Visual Studio Code and broad support for working with files and data, including tools for structured output, text conversion from PDF, table searching, and even running LLM agents and tools in containers. GenAIScript also supports GitHub Models, LocalAI, and security features for content. It expands capabilities for developers to effectively automate and optimize tasks using the latest AI models.
