# Desktop Shortcut
> Quick access to a specific function or page – and it's equally useful on a computer and on mobile.

A desktop shortcut ([desktop shortcut](https://sharons-shortcuts.ie/creating-a-desktop-shortcut-in-windows/), [alias](https://support.apple.com/en-ng/guide/mac-help/mchlp1046/15.0/mac/15.0), [symlink](https://en.wikipedia.org/wiki/Symbolic_link), [web clip](https://www.hexnode.com/blogs/explained/what-are-web-clips-on-ios/#:~:text=Web%20Clips%2C%20not%20to%20be,for%20iPadOS%20and%20macOS%20devices.), [web link](https://medium.com/@appdevinsights/app-links-web-links-deep-links-universal-links-and-dynamic-links-06ce844ee891#:~:text=2.-,Web%20Links,-Web%20Links%20are)) is a shortcut for opening internet addresses, files, or applications. It is one of the oldest and simplest ways to improve usability.

On a computer, it is usually an icon on the desktop or in the dock that opens a specific application or webpage with a single click. For mobile devices or web applications, however, adding shortcuts has slightly different rules and, at the same time, greater potential.

On desktop computers, creating a website shortcut is easy. Most browsers offer the option *Add to Desktop* or *Create Shortcut*. The user doesn't have to dig through bookmarks or search – just click and they're there. For web applications that behave like native programs, this significantly improves the impression and comfort.

On mobile, shortcuts have a similar meaning – but they need to be handled with more care. On an iPhone, you can use the *Add to Home Screen* function in the Safari browser to create an icon that looks and behaves like a regular app. On Android, the Chrome browser offers the *Add to Home Screen* option – it often even offers this automatically if the web app is properly prepared.

Users almost don't distinguish between an app and a website opened from a desktop shortcut – and that's exactly what we're aiming for if we want them to return to the website regularly.

Therefore, it's important not to leave it to chance. Notifications, instructions, or automatic offers to create a shortcut can bring more returns and a better experience. The main thing is to be prepared for it: have a responsive design, a correctly filled manifest, an icon for the mobile home screen, and, most importantly, a clear motivation for why the user should return.
