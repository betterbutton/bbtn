# Ntfy.sh – send notifications from the web
> Ntfy.sh is a simple service for sending notifications. To send a notification, a simple HTTP request is enough. Reception is provided by a ready-made mobile app.

![Screenshot of the ntfy.sh website.](https://substack-post-media.s3.amazonaws.com/public/images/4bef6c9d-1aa0-43bf-bc2a-7554e874e741_1895x1064.png "542x304")

ntfy will send you a notification to your mobile or browser

[Ntfy.sh](https://ntfy.sh/) is an open-source service that allows you to easily send push notifications to mobile devices, desktop, or web. It's a simple tool for sending messages via HTTP requests without complex configuration or dependence on external services. You can send a notification even without a server using JavaScript.

You can send notifications from your own website, and it also works great together with [the previously recommended service cron-job.org](https://www.reknisioweb.cz/p/cron-job), which runs scripts on the website at specified intervals.

Ntfy.sh delivers notifications to Android, iOS, and the web interface. You can customize notifications – set custom channels, priorities, and content. And importantly, ntfy also offers message encryption, which ensures security.

The service provides a free tier without registration or a subscription for a few dollars. If you want your own installation, you can take advantage of the fact that [ntfy is open-source](https://github.com/binwiederhier/ntfy).

Developers can use ntfy to send alerts for errors and build results, server administrators to monitor outages, or home automation to send notifications from sensors or smart devices.

It's simple, flexible, and requires no registration or any complex cloud services. Just send a request and the notification will immediately appear on your device.
