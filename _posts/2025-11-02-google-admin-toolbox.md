# Google Admin Toolbox offers quick diagnostics not only for Google Workspace
> Google Admin Toolbox – a web-based toolkit for testing, diagnostics, and troubleshooting domain and Google Workspace service issues.

![Google Admin Toolbox home screen](https://substack-post-media.s3.amazonaws.com/public/images/a41b8960-7465-4f76-8e0e-a85bb0147440_1897x1079.png "606x344")

[Toolkit for Google administrators](https://toolbox.googleapps.com/apps/main/?hl=cs) ([Google Admin Toolbox](https://toolbox.googleapps.com/apps/main/?hl=en)) is a free online toolkit that helps administrators quickly identify and resolve domain issues not only in Google Workspace environments. It offers simple web tools for connection testing, [DNS record verification](https://toolbox.googleapps.com/apps/dig/), [log analysis](https://toolbox.googleapps.com/apps/loganalyzer/), and tracking [email delivery issues](https://toolbox.googleapps.com/apps/messageheader/) – all without requiring any installation.

The foundation of the entire toolkit is the [Check MX](https://toolbox.googleapps.com/apps/checkmx/) tool, which verifies mail and DNS records, flags configuration errors, and displays mail delivery status using a clear color indicator – green, yellow, or red. Another useful helper is [Messageheader](https://toolbox.googleapps.com/apps/messageheader/), which can parse email message headers, display delivery paths, and reveal why delivery is taking too long or messages are delayed.

For debugging browser and network issues, [Browserinfo](https://toolbox.googleapps.com/apps/browserinfo/) serves its purpose. This tool collects browser setting information, measures response times when communicating with Google services, and checks whether the connection is blocked, for example, by a firewall. Similarly, [User‑agent](https://toolbox.googleapps.com/apps/useragent/) helps determine which browser a user is using and whether any incompatibilities are occurring. Both tools are useful even outside the Google Workspace environment.

The [Log Analyzer](https://toolbox.googleapps.com/apps/loganalyzer/) and [Log Analyzer 2](https://toolbox.googleapps.com/apps/loggershark/) tools speed up the search for error causes. They can read logs from Google services, such as Chrome, Google Workspace Sync for Microsoft Outlook, or Cloud Directory Sync, and within seconds highlight the problem and explain its cause.

Other useful add-ons, like [Encode/Decode](https://toolbox.googleapps.com/apps/encode_decode/) for encoding and decoding text from various formats (Base64, Quoted-Printable, URL encoding…), or [Screen recording](https://toolbox.googleapps.com/apps/screen_recorder/) for capturing on-screen activity, help with more complex situations and are good to have on hand, perhaps in favorites.

For administrators who need to keep their Google Workspace environment running quickly and reliably, [Google Admin Toolbox](https://toolbox.googleapps.com/apps/main/) is an indispensable helper.
