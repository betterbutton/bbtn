# How to Use a Screen Reader on an Inaccessible Website
> 

„A screen reader essentially removes all visual layout, lines everything up, and then navigates from start to finish.“\
**— [Craig Abbott](https://www.craigabbott.co.uk/blog/what-is-it-like-to-use-a-screen-reader-on-an-inaccessible-website/)**

Using a screen reader on an inaccessible website is like walking through a dark room with only a narrow flashlight beam. [Screen readers transform the web into a linear flow](https://www.craigabbott.co.uk/blog/what-is-it-like-to-use-a-screen-reader-on-an-inaccessible-website/), which makes it difficult for users to orient themselves in a 2D space. If a website is not properly structured, navigation becomes frustrating. Dynamic content can be problematic – excessive updates overwhelm users or provide unclear information. For better accessibility, it is crucial to focus on meaningful content descriptions (ALT text, icon labels) and think about the purpose of elements.
