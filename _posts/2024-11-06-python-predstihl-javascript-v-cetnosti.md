# Python overtook Javascript in usage frequency on GitHub
> 

![Graph shows that Python overtook Javascript in usage on GitHub. Java is also declining, as is PHP](https://substack-post-media.s3.amazonaws.com/public/images/0170d846-43c6-41eb-9875-dec6543f527d_1000x1000.jpeg "354x354")

[Top Programming Languages on GitHub](https://github.blog/news-insights/octoverse/octoverse-2024/)

According to [current statistics: Python has overtaken Javascript on GitHub as the most used language](https://github.blog/news-insights/octoverse/octoverse-2024/) in usage frequency and probably AI is to blame, where Python is better applied and various demos are well prepared with it, for example in Jupyter notebooks.

Also interesting is the steady decline in PHP's popularity, which in the last two years, as if it has already stopped. Can AI help to make Python the new PHP? Will a replacement for WordPress, Magento, phpBB or Moodle emerge in Python?
