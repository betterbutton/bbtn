# Approaches to Light and Dark Mode in CSS
> 

The article [Come to the light-dark() Side](https://css-tricks.com/come-to-the-light-dark-side/) by Sara Joy focuses on implementing light and dark modes on websites using modern CSS features like `color-scheme` and `light-dark()`. These tools enable automatic switching of color schemes based on user or operating system preferences, without complex `@media` queries.

Sara describes various ways to define colors, including mixing with `color-mix()`, and shows how to store user preferences using `localStorage`. She recommends offering users a choice between light, dark, and automatic modes, emphasizing accessibility and comfort for diverse user needs.
