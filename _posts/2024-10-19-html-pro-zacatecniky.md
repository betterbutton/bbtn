# HTML for beginners
> 

What better time to start learning something new than on the weekend! This web guide provides [HTML basics for people with no prior coding experience](https://htmlforpeople.com/). Blake Watson presents HTML as a universal format for web documents, which is accessible to everyone, not just professional developers.

Step by step, you will learn how to create a simple website from scratch, without the need for special programs. The guide emphasizes that HTML is a robust format with a long history and a simple start. The e-book is designed to be available for users of all major operating systems, with an emphasis on using free tools.
