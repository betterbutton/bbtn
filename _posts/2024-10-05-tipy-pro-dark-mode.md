# Practical Tips for Dark Mode Design
> 

Dark mode has become a popular feature that improves user experience by reducing eye strain, saving battery, and offering a modern look. Mallikarjun Pasupuleti wrote a [practical guide](https://medium.com/@mallikarjunpasupuleti) on color selection and accessibility when implementing dark mode.

It is essential to reduce brightness, choose contrasting colors, and ensure text readability in both modes. [Dark mode not only aesthetically enhances applications](https://medium.com/@enzoymribeiro/designing-in-the-dark-the-best-practices-for-dark-mode-ui-0ffe20057311), but also allows for better adaptation to user preferences.
