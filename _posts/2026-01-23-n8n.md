# N8n is no-code automation you can own
> An automation platform that combines visual, no-code building with the control of developer tools, designed for automation you can own and develop long-term.

![Screenshot of the n8n editor workflow, where an AI agent processes incoming chat messages, uses OpenAI and tools, and then routes success or failure messages to Slack.](https://substack-post-media.s3.amazonaws.com/public/images/02ba7f41-c2a3-428b-b378-ba06eefa8cb2_1024x594.png "592x343")

To set up a [n8n workflow](https://n8n.io/), there are many [step-by-step guides](https://hatchworks.com/blog/ai-agents/n8n-for-workflow-automation/) available online.

*[N8n](https://n8n.io/)* (pronounced: n-eight-n) is a workflow automation platform that connects services, moves data, and executes logic using a visual, node-based editor. The name n8n is short for *nodemation*, or *node‑based automation*. At first glance, it seems like a tool that doesn't require writing code. In reality, however, it behaves like part of your infrastructure. It is precisely this combination that is key. It allows for a quick start while also handling automation in a production environment.

The fundamental principle is workflows composed of nodes and their connections. Each execution processes data step by step, with options for branching, looping, and transformations. The graphical model allows you to gradually move from simple tasks to automation that runs continuously and supports the daily operation of websites and services.

An important feature is the ability to run on your own server. This is crucial when automation works with sensitive data, access credentials, or internal systems. Automation thus becomes not another external service, but a controlled part of the technical infrastructure.

N8n offers an extensive ecosystem of integrations and is gradually incorporating artificial intelligence into both the creation and execution of workflows. Emphasis is placed on reliability, security, and long-term sustainability.

It works best where automation is important, will evolve, and is worth owning. For one-off or trivial tasks, there are simpler paths. For everything else, n8n is a tool that lasts.
