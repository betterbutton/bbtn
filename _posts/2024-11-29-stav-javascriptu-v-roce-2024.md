# State of JavaScript 2024
> 

JavaScript is undergoing an evolution that could change the way we write it in the future. New [TC39 proposals](https://tc39.es/) bring innovations, and some are starting to appear in browsers. The popularity of [TypeScript](https://www.typescriptlang.org/) also opens the possibility for native type support.

The current version of JavaScript is still familiar, but in five years, everything could be different. The community, including you, can influence development, for example by participating in the [State of JavaScript 2024](https://survey.devographics.com/en-US/survey/state-of-js/2024) survey.
