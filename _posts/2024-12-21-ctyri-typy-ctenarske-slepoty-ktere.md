# Four Types of Reader Blindness That Influence User Attention
> 

> “Your user interface should reflect the hierarchy of importance, support scannability, and facilitate orientation.”\
> **— [Krisztina Szerovay](https://uxknowledgebase.com/selective-attention-content-blindness-in-ux-design-e53033b597c6)**

User attention is a limited resource that UX designers must carefully account for. [Krisztina Szerovay explores four types of blindness](https://uxknowledgebase.com/selective-attention-content-blindness-in-ux-design-e53033b597c6) that can negatively impact the user experience: content blindness, banner blindness, change blindness, and inattention. The key is to create a visual hierarchy that naturally guides users to the most important information without being overwhelming. Practical tips include avoiding visually identical elements with different functions and minimizing cognitive load. Emphasis is placed on clarity and consideration of user goals and motivations.
