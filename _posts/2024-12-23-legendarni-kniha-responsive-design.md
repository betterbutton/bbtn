# The legendary book Responsive Web Design is now freely available to read
> 

![Cover of the book Responsive Web Design](https://substack-post-media.s3.amazonaws.com/public/images/7e27c563-06f0-470c-81c0-2475b0f77579_301x462.png "301x462")

Ethan Marcotte launched a revolution in May 2010 with the article [Responsive Web Design](https://alistapart.com/article/responsive-web-design/) in the online magazine A List Apart. His concept of websites that adapt to the device you are currently viewing them on was based on three ideas: fluid layout ([fluid layout](https://www.geeksforgeeks.org/understanding-fluid-layouts-in-web-design-how-to-use-it/)), responsive image sizes, and [media queries](https://www.vzhurudolu.cz/prirucka/css3-media-queries) in CSS with viewport width support.

A year later, he published a book under the same name, and responsive design became a big thing that significantly reduced web development costs. Until then, if web developers were working on designs for different devices, they worked with [adaptive design](https://en.wikipedia.org/wiki/Adaptive_web_design). This realistically meant creating and maintaining several alternative versions of the website for different devices.

The book was published by A Book Apart, which ceased operations this year and returned licenses to authors for their books, including source files. Ethan decided to make the book Responsive Web Design, which is one of the cornerstones of the modern approach to web design, [available to readers on the web for free](https://ethanmarcotte.com/books/responsive-web-design/).
