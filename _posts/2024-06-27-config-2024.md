# Figma has news: editor redesign, AI, presentations, improved Dev mode and FigJam
> 

At yesterday's ConFig 2024 conference [Figma presented several product updates](https://www.figma.com/whats-new/):

**[Artificial intelligence in the editor](https://www.figma.com/ai/)** – AI will help designers to design, turn designs into prototypes, name layers, generate and translate design content, search by image, remove background from image (Finally we [don't have to do it in Excel](https://support.microsoft.com/en-us/office/remove-the-background-of-a-picture-in-office-c0819a62-6844-4190-8d67-6fb1713a12bf#:~:text=Select%20the%20picture%20that%20you,you%20have%20selected%20a%20picture.)!)

**[Redesigned UI3 editor](https://www.figma.com/blog/behind-our-redesign-ui3/)** – the new interface cleared the canvas by converting tools into floating palettes that expand contextually.

**[New Dev mode and Code Connect](https://www.figma.com/community/file/1234941143610339388)** – improvement in communication between designers and developers using flags (modified, completed). Option to attach source code to a visual component.

**[Figma Slides](https://www.figma.com/slides/)** – an alternative to Keynote and PowerPoint, currently free.

**[Pages in FigJam](https://help.figma.com/hc/en-us/articles/24005082123159-Create-and-manage-pages-in-FigJam)**

You can watch the entire event [from the recording on YouTube](https://www.youtube.com/watch?v=NW-dUfXIVvk&list=PLXDU_eVOJTx6MtJL_3HAmsN2rKoSLvM_g).
