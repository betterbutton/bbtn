# Notify Me alerts you to changes on the web
> Notify Me monitors changes on web pages and informs you about them via email. For free, it checks the specified address three times a day, in the paid plan every quarter hour.

![Screenshot of the Notify Me website.](https://substack-post-media.s3.amazonaws.com/public/images/1cf8094a-c80d-49b1-a1f9-47792643c6b5_1899x952.png "1456x730")

Notify Me monitors the specified address and alerts to changes

Sometimes it's useful to have the ability to track changes on web pages and be informed about them. When a page doesn't send notifications to mobile apps, doesn't distribute newsletters, and doesn't even have an RSS feed, all that's left is to set up monitoring on it. Such a page is, for example, our [overview of professional events and conferences](https://www.reknisioweb.cz/p/konference-2025). There are plenty of services that track changes on web pages.

We chose [Notify Me](https://notify-me.rs/). For free, it will check changes on three addresses three times a day. If you need to track more addresses or more frequently, you can buy either a lifetime or monthly subscription. The service will inform you about changes via email.
