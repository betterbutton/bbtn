# How to Create Paper Wireframes
> 

A wireframe is a visual guide that serves as a blueprint for user interface design. It is used to sketch the basic structure of web or mobile applications. Wireframes help designers and developers visualize the layout and functionality of various interface elements without the distracting details of graphics or colors.

It is a tool in the UX design process that allows for quick testing and refinement of ideas, leading to the creation of effective and user-friendly products. A specific subtype is the paper wireframe, drawn with a pencil or pen on torn pieces of paper, which can then be elevated into an interactive paper prototype. Paper wireframes help create effective and user-friendly screens.

Roberto Lisandro will guide us through creating paper wireframes practically in four steps:

1.  **Make a list of necessary elements,** for example, for a fitness app that includes screens like Home, Workouts, Community, and Profile.

2.  **Create several different versions of the element layout.**

3.  **Select suitable elements from each version.**

4.  **Combine these elements into one final wireframe.**
