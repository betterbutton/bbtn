# GSAP: Animations That Will Move Your Web
> A powerful library for JavaScript animations that developers use to create smooth and interactive user interfaces.

![Screenshot of GSAP website's landing page.](https://substack-post-media.s3.amazonaws.com/public/images/b0257a9a-0810-4620-8fd2-fbfaa1c5a9ba_1896x1079.png "562x319")

[GSAP](https://gsap.com/) – animations for the web

[GreenSock Animation Platform](https://www.greensock.com/), known as [GSAP](https://gsap.com/), is a powerful JavaScript library that enables the creation of fast, smooth, and perfectly controllable animations on the web and in mobile applications. Thanks to its reliability and flexibility, it has become a popular tool among developers working on modern user interfaces.

GSAP allows animating not only CSS properties but also SVG elements, canvases, or any arbitrary objects. It offers much broader possibilities than standard CSS transitions or animations.

One of the popular tools is the [ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/) plugin. It allows linking animations to page scrolling, without the need for heavy third-party libraries. Thanks to it, you can easily create parallax effects, transitions during scrolling, or gradual content reveals.

GSAP also offers timelines, which significantly simplify the control of complex animation sequences. Developers thus have clear control over what happens when, and the code remains clear even in extensive projects.

The library is also optimized for high performance. It limits unnecessary repaints and saves memory, so the resulting applications remain smooth even on older devices.

Thanks to clear documentation and an active community, getting started with GSAP is easy. Developers who already know JavaScript quickly get oriented and can start creating without unnecessary delays.

If you are building a modern web interface and care about performance and precision, GSAP is a reliable foundation to build upon.

And the best for last: The library and its creators were adopted by the WebFlow platform, allowing them to cancel their price list and offer [all the library's features for free](https://gsap.com/pricing/).
