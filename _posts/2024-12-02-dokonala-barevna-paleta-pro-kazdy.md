# Perfect colour palette for every project
> 

Creating a colour palette is crucial for design consistency and effective work. Colour scales, meaning the spectrum of shades from light to dark, ensure consistency and easy usability in every project. Sanketh Sharathkumar shows us [how to do it](https://www.fourzerothree.in/p/designing-colour-palettes-that-work).

It starts with defining basic colours (brand, accent, grey, feedback) and their shades using the HSB model, where changes in brightness and saturation create light and dark variations. For grey shades, lower saturation is used to harmonize with the main palette.

Tools like [Colorbox](https://colorbox.io/) or [Tailwind CSS palettes](https://tailwindcss.com/docs/customizing-colors) make the process easier, but manual tuning remains necessary. The result is a flexible and accessible design.
