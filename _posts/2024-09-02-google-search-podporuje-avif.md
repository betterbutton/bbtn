# Google Search Now Supports AVIF
> 

The last significant argument against using the [AVIF](https://en.wikipedia.org/wiki/AVIF) image format has fallen. [Google announced support for the AVIF format](https://developers.google.com/search/blog/2024/08/happy-avifriday) in its search, which includes Google Images and other areas where images are used. AVIF is an open format based on the [AV1](https://en.wikipedia.org/wiki/AV1) compression standard. It is now compatible with Googlebot without requiring special indexing measures. However, before switching to AVIF, website owners should carefully consider whether this format is suitable for their needs and potentially set up server redirects if file names change.

In February, after Chrome, Firefox, and Safari, [Edge also started supporting AVIF](https://www.reknisioweb.cz/p/05#:~:text=Edge%20kone%C4%8Dn%C4%9B%20podporuje%20AVIF!), so nothing prevents the use of the new format on websites. If you want to learn more about the AVIF format, [read Up and Down](https://www.vzhurudolu.cz/prirucka/avif).
