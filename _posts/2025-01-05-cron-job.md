# Cron-job.org regularly visits a website or runs a script
> A free web utility cron-job.org at a pre-set interval for a web address. Just like the Linux cron daemon, but on the web.

![](https://substack-post-media.s3.amazonaws.com/public/images/7c650e11-ed50-42af-83cc-25cfde619f03_2532x1882.png "1456x1082")

Cron-job.org – a web utility for repeated tasks

Using the service [cron-job.org](https://www.cron-job.org) you can set visiting a web address at a regular interval. As the name suggests, this utility works similarly to the Linux [cron](https://www.root.cz/man/8/cron/). You set the URL and interval, and cron-job.org will send an HTTP request to that address at the specified interval.

Unlike the Linux cron, which sometimes [confuses](https://crontab.guru/) even seasoned geeks when setting up, cron-job.org can be set up by anyone. Cron-job.org sends tens of millions of requests every day. You can have it execute a task every minute or just once a year. In the free plan, you can use it virtually without limits, and if you decide to contribute to its operation, the operators leave the amount of the contribution up to you.

What is such a service good for? If you don't have access to cron on the server, you'll appreciate that it can call your scripts. Maybe you need to send an email with an overview of new articles every day. Maybe you want to regularly check stock prices. Generate a report. Prevent a service from being canceled due to inactivity. Monitor website functionality. Start synchronization or backup. Kick off an automation workflow. Think about what you would need. Let your imagination run wild.

[There are many web crons](https://pinboard.in/u:garcon/t:cron/t:tools) ([Cronitor](https://cronitor.io/), [FastCron](https://www.fastcron.com/), [EasyCron](https://www.easycron.com/)) and you can even find [completely free services](https://www.cronjob.de/) among them, but [cron-job.org](https://cron-job.org) belongs to those that reliably do what they are supposed to, and you don't have to be a Linux administrator to be able to use it.
