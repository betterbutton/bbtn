# Claude AI not only programs but also saves to the repository
> 

[Model Context Protocol](https://modelcontextprotocol.io/introduction) presents an open standard for [connecting AI tools with data sources](https://www.anthropic.com/news/model-context-protocol), thereby removing barriers caused by information isolation. Model Context Protocol enables two-way communication between systems and AI applications thanks to a unified protocol that replaces fragmented integrations.

Developers can use pre-configured servers for platforms like Google Drive or Slack, or create their own. Model Context Protocol has already been adopted by companies like Block and Apollo, which appreciate its benefits for innovation and efficiency. The project is open-source, supports collaboration, and promises to simplify AI access to relevant data across systems.
