# A font you don't write with, but animate with.
> 

Valdemar Erk decided to use the [HarfBuzz](https://en.wikipedia.org/wiki/HarfBuzz) technology, which we have already encountered with the [LLaMa.ttf](https://www.reknisioweb.cz/p/llama-font) font, which has artificial intelligence built into it. Instead of ligatures, for which HarfBuzz is designed, he modified it so that while you type, instead of letters, an [animation is displayed](https://youtu.be/Ms1Drb9Vw9M?si=JQDaVcvrRDDM52Pb&t=1000) on the screen. By typing, you control the animation, so you can [play Tetris with the font, for example](https://www.youtube.com/watch?v=Ms1Drb9Vw9M&t=1376s).
