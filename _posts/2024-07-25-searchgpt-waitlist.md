# ChatGPT gets search
> 

![SearchGPT search box asks: What are you looking for?](https://substack-post-media.s3.amazonaws.com/public/images/bea86f46-6944-49b5-a33c-794ed2f03b06_1360x766.jpeg "1360x766")

[Are you already on the waitlist?](https://chatgpt.com/search) For some time now, SEO specialists have been reporting that websites are being crawled by OpenAI's bot, so the arrival of an AI-based search engine was more than expected. "They confirm that this time OpenAI really crawled all websites where they monitor crawling and some very intensively," [wrote on LinkedIn](https://www.linkedin.com/feed/update/urn:li:activity:7222318900619706371/) web analyst Marek Lecián.

OpenAI describes [SearchGPT as an AI prototype with search](https://openai.com/index/searchgpt-prototype/), which provides sourced answers. In addition to search, SearchGPT offers real-time website browsing. It promises to provide answers not only in text but also in visual and video formats. It should thereby [approach the ideal of a responder](https://www.reknisioweb.cz/p/ai-serp), which, instead of a list of links, provides a clearly formulated and sourced answer to the query.

On a similar principle works [Perplexity](https://www.perplexity.ai/), which raises concerns that it does not respect robots.txt, because it is not considered a full-text search engine, but an agent acting on behalf of a human.
