# Glassmorphism
> A UI design trend using blur and transparency to mimic the look through frosted glass.

Glassmorphism ([glassmorphism](https://www.interaction-design.org/literature/topics/glassmorphism)) is a popular visual style that creates the impression of translucent layers and a blurred background. It looks like frosted glass and helps create a sense of depth and visual structure. This allows the user to better perceive important elements without losing track of what's happening in the background.

This style spread with the arrival of [Fluent Design](https://fluent2.microsoft.design/) from Microsoft and with the update of [macOS Big Sur](https://www.apple.com/cz/newsroom/2020/06/apple-introduces-macos-big-sur-with-a-beautiful-new-design/) from Apple. Both systems introduced graphical elements that seemed to float above a blurred content. In practice, designers use properties like background blur and transparent colors. The effect is complemented by subtle borders and shadows that improve readability.

[Glassmorphism](https://dribbble.com/search/glassmorphism), however, is not just about visual impression. It also has practical advantages. It helps focus attention on the main content while the background remains partially visible. On the other hand, it can strain performance, especially on mobile devices. Using blur requires more powerful graphics, and not all browsers support it. Therefore, it is necessary to design an alternative look for older devices.

[Glassmorphism](https://ui.glass/generator/) should be used in moderation. If overused, it can worsen readability and accessibility. The designer should always consider how well the content is visible and understandable.

Overall, [glassmorphism](https://www.figma.com/community/plugin/1197106608665398190/glassmorphism) is a powerful visual tool. When used judiciously, it can make an interface look modern, clear, and elegant. It is a style that can improve the user experience, provided we do not forget technical limitations and user needs.
