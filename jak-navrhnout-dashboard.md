# How to design a clear and effective dashboard
> 

To [create an effective dashboard](https://www.geckoboard.com/best-practice/dashboard-design/), the main thing is to clearly define the dashboard's goal and maintain simplicity and clarity. Consistent layout ensures better orientation and [simplicity simplifies complex information into a clear form](https://towardsdatascience.com/12-less-obvious-rules-for-your-dashboard-4d7c35a1cee9). The purpose of the dashboard is to allow the user to [quickly interpret data](https://medium.muz.li/10-rules-of-dashboard-design-f1a4123028a2), without having to expend effort on understanding individual elements.
