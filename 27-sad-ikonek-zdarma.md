# 27 Great Free Icon Sets for Your Project

Dmitriy Bunin from [Frames X](https://framesxdesign.com) UI design kit for Figma did the work and researched free icon sets, sorted them alphabetically and commented on them. Here are the [27 sets](https://framesxdesign.com/learn/best-free-icon-sets) he recommends for your next project:

Some sets have thousands of items, others only a few hundred.

Dmitriy also offers criteria for choosing an icon set for UI design, which a designer should focus on:

1.  **Personality and branding:** Icons should match the brand's visual identity and harmonize with other design elements.

2.  **User-oriented context:** Consider how and when users will interact with the icons, and choose a style (solid, outlined, duotone, etc.) based on the context.

3.  **Consistency:** Maintain uniformity across all project icons, including the same spacing and visual weight.

4.  **Icon style:** Choose between solid and outlined icons based on project needs, with outlined being suitable for modern and data-dense interfaces.

5.  **Stroke weight:** Ensure the stroke weight of the icons matches the size and style of the fonts.

6.  **Number of icons and adaptability:** The set should contain enough icons for the entire project and allow customization according to design needs.

The chosen icons should be high quality to avoid the need for frequent redesigns.
