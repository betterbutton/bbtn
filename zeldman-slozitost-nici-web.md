# Zeldman: How Complexity Destroys the Web and What to Do About It
> 

Jeffrey Zeldman [criticizes the excessive complexity of modern web design](https://alistapart.com/article/cult-of-the-complex/), which often prioritizes technical innovations over the real needs of users. He points out that web architecture is becoming unnecessarily complicated, leading to the excessive use of frameworks and tools that developers often don't fully master.

> „Simplicity means doing the hard work that makes experiences appear seamless.“\
> **— Jeffrey Zeldman, [The Cult of the Complex](https://alistapart.com/article/cult-of-the-complex/)**

Zeldman calls for a return to the basic principles of simplicity, semantic HTML, and clear code that is user-friendly and developer-friendly. In conclusion, he warns against the danger of forgetting the true goals of web design: improving user experience and effective communication.
