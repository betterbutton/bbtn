# Export from Search Console to BigQuery
> 

Marek Prokop [recommends exporting outputs from Google Search Console to BigQuery](https://www.linkedin.com/feed/update/urn:li:activity:7218313304132173824/). You will then be able to compare development over time. The export is available for a little over a year, and if you started exporting from the beginning, you already have the possibility to make yearly comparisons. If you haven't started exporting yet, according to Marek, it's high time.
