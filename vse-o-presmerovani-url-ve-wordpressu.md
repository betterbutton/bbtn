# All about URL redirection in WordPress
> 

WordPress automatically preserves the original URLs of articles when addresses are changed, which is useful for SEO and accessibility. If needed, old redirects can be displayed and removed using the Slugs Manager plugin, [as Jan Polzer writes on his blog](https://www.maxiorel.cz/jak-na-redirekty-ve-wordpressu-mazani-tech-automatickych-vytvareni-vlastnich). Creating custom redirects is possible in the Rank Math SEO plugin, which offers customizable redirects, including the choice between permanent (301) and temporary (302, 307) redirects. Proper setup of redirect codes is crucial for SEO and the sustainability of web content, for example, during a temporary campaign or a permanent page move.
