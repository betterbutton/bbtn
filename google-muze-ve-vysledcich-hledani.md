# Google can display og:title in search results
> 

Google has updated its documentation for developers to add information that the search engine [Google can now use the content of the og:title meta tag](https://developers.google.com/search/docs/appearance/title-link#sources) to create titles for search results. [Search Engine Roundtable notes](https://www.seroundtable.com/google-title-og-title-37973.html) that this tag is typically used for titles when sharing on social networks, but now Google can also use it for the link title in search.

With this step, Google has expanded the list of sources from which it automatically generates titles, and in addition to classic `title` elements or `h1` headings, it now also includes the content of `og:title` meta tags.
