# Practical Guide to UX Flowcharts
> 

Creating UX flowcharts helps define user journeys and prevent misunderstandings in product design. Marianne Braum shows us [how to communicate effectively using flowcharts](https://uxplanet.org/demystifying-ux-flowcharts-visual-vocabulary-c81a8e9ad5b7). A well-structured flowchart not only reduces the risk of redesign but also provides a clear overview of the screens and functions that will be needed.

The [Visual Vocabulary](http://jjg.net/ia/visvocab/) created by Jesse James Garrett offers architects and designers standardized elements, such as pages, connectors, and regions, which simplify the creation and sharing of designs. Key are decision points and conditional paths that allow for creating different scenarios based on user actions or permissions.
