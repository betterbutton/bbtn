# Google Translate Can Handle a Hundred More Languages
> 

Polyglot [Martin Hassman noticed](https://x.com/hassmanm/status/1807445925318525136), that Google Translate, the web translator, has added 110 new languages to its collection.

> "The classic problem with online translators is supporting a small number of languages; as soon as you go a bit off the mainstream, you're out of luck."
> **— [Martin Hassman](https://x.com/hassmanm/status/1807445925318525136) on Twitter**

Now you can translate from Portuguese, Romanian, Ossetian, Venetian, Sicilian, Occitan, Tibetan, Punjabi. Only for Martin's favorite language [toki pona](https://cs.wikipedia.org/wiki/Toki_pona) we still have to wait.
