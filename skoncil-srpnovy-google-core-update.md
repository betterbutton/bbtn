# August Google Search Core Update Has Concluded
> 

Pavel Ungr on Twitter [announces](https://x.com/PavelUngr/status/1831074288423817591) that according to the announcement from Google Search Central, the [large August search update](https://x.com/googlesearchc/status/1831046515374330114) on Google has already concluded. The search engine behavior should now be stable, so it's the best time to go and tally up gains and losses in search results.
