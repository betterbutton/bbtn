# Do variable fonts save data compared to regular fonts?
> 
How do variable fonts perform? Stoyan Stefanov conducted an analysis using variable fonts with variable weight `[wght]` and compared them with regular fonts. The results showed that variable fonts that combine different cuts (e.g., normal and bold cut) in one file, [are slightly smaller than two separate font versions](https://www.phpied.com/web-font-file-size-study-a-variable-font-addition/).

> „We likely save due to duplicate metadata and some other elements in two files versus one.“\
> — **[Stoyan Stefanov](https://www.phpied.com/web-font-file-size-study-a-variable-font-addition/)**

The median file size of the variable font was 34,744 bytes, which is smaller than the sum of two standard versions. The benefit also lies in a lower number of HTTPS requests.
