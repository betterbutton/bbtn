# Case Study: Application for a More Pleasant Library Visit
> 

Psychologist Sena Dökmeci [designed an application that would facilitate orientation and access to libraries](https://medium.com/design-bootcamp/case-study-designing-the-library-visitor-experience-648ea1fdfb76). Through interviews and research, she identified key user needs – determining the occupancy of study spaces, easy book search, and information about library operating hours.

The resulting application, named The Library, allows viewing current library occupancy and access to their databases. The prototype was created using Figma with an emphasis on simplicity and clarity.
