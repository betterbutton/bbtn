# Open source alternative for automation
> 

Do you prefer open source and self-hosted solutions over public services like [Make](https://www.make.com/en), [IFTTT](https://ifttt.com/), or [Zapier](http://Zapier.com)? Then [Automatisch](https://automatisch.io/) might come in handy. [Download it from Github](https://github.com/automatisch/automatisch/) and install it as Docker. Then it works just like you're used to from commercial services: A trigger invokes an action. Actions can be chained in workflows.
