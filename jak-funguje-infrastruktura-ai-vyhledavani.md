At the Config 2024 conference, Figma introduced AI search, which allows you to easily find designs and components using a text description, screenshot, or layer selection. This system uses embeddings generated by the CLIP model to efficiently index and search for visual and textual similarities.

In the article [The infrastructure behind AI search in Figma](https://www.figma.com/blog/the-infrastructure-behind-ai-search-in-figma/), developers Maggie Zhou and Isaac Goldberg explain what the infrastructure of the new AI search looks like.

The development involved solving challenges such as scalability, efficient index management, and cost optimization. Techniques like vector quantization and infrastructure optimization using a headerless server and parallel processing on SageMaker were deployed. Thus, Figma enables fast and accurate search even in large and complex design systems.
