# Step by Step to Free Web Hosting on GitHub
> 

Raushan Raj wrote a brief guide that will step-by-step show you [how to host a static website on GitHub](https://medium.com/@raushanraj9950/host-your-html-css-and-js-website-on-github-3176b3715893). It assumes you already know how to upload code to GitHub (e.g., via the web interface), but it covers the process from creating a folder to successful publication.

First, you will upload the project to GitHub, set the index.html file in the main folder, and enable [GitHub Pages](https://pages.github.com/) in the settings. After a short while, the website will be accessible via the link GitHub generates. This method offers an easy way to publish for those seeking a quick, cost-free solution.
