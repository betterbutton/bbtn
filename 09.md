# Linkodrome

**No one in the Czech Republic can monitor the DSA.** We have written about the [Digital Services Act](https://reknisioweb.substack.com/p/06) and its [impacts](https://reknisioweb.substack.com/p/07) several times. [Lupa points out](https://www.lupa.cz/clanky/co-prinasi-nova-regulace-online-diskusi-akt-o-digitalnich-sluzbach-v-praxi/) that the Ministry of Industry and Trade has not even been able to prepare a Czech [implementing law](https://odok.cz/portal/veklep/material/ALBSCWAFVK4T/) a year and a half after the regulation was approved, which would [grant ČTÚ the authority](https://ctu.gov.cz/evropske-narizeni-o-digitalnich-sluzbach-dsa) to oversee compliance with the regulation.

**Do you have a computer security expert?** And could we see them? The [NIS2 cybersecurity directive](https://osveta.nukib.cz/course/view.php?id=145) will most likely apply to you as well. It turns out we [have too few experts](https://www.lupa.cz/clanky/regulace-podle-nis2-ministri-jsou-proti-koncentraci-moci-v-rukou-nukibu-oboru-chybi-odbornici/).

The boy replies: "But honey, it's code outside."](https://substack-post-media.s3.amazonaws.com/public/images/cce6cb51-6f82-4f9e-ab1d-39f9c195e161_640x480.webp "640x480")

It's raining and it's getting dark outside...

**A Book Apart is ending book publishing after 14 years.** In the [first newsletter](https://reknisioweb.substack.com/p/01) we ranked A Book Apart first among recommended publishers. All the more we were struck by the news that no [more books will be published](https://abookapart.com/blogs/press/a-new-chapter-for-a-book-apart) under that brand. If you wanted to buy [any book from their catalog](https://www.goodreads.com/list/show/13967.A_Book_Apart), hurry before they sell out. And while you're in that bookstore, don't forget to give the book five stars and write a short review, [asks Erika Hall](https://www.linkedin.com/posts/erikahall_if-you-are-in-the-mood-to-do-a-favor-now-activity-7170534222284120064-x4-H/), author of our favorite [book on user research](https://www.goodreads.com/book/show/17236175-just-enough-research). Tell us in the comments which of their books is your favorite!

**Google has a new login page.** The [login page](https://gizmodo.com.au/2024/02/googles-login-page-got-a-redesign-heres-a-first-look/) has been dressed in Material Design 3 components. Honestly, we expected a bolder shift towards more modern login methods – like [passkeys](https://www.irozhlas.cz/veda-technologie/technologie/passkeys-prihlasovani-bez-hesla-kyberneticka-bezpecnost_2305061503_ara), especially since [all browsers have supported them](https://caniuse.com/passkeys) for a year and a half now.

**Seznam.cz will start charging people for irrelevant ads.** Well, that's how it is. You'll either be able to browse Seznam.cz for free with ads targeted specifically at your profile, or [for money with completely irrelevant ads](https://www.lupa.cz/aktuality/seznam-cz-zacne-testovat-platby-za-necileni-reklamy-bude-za-to-chtit-desitky-korun-mesicne/).

**Word's default font is no longer Calibri.** Microsoft has [changed the default font](https://edition.cnn.com/2024/02/22/us/microsoft-default-font-aptos-calibri-cec/index.html) in Word to Aptos. [Typography enthusiasts are pleased](https://www.linkedin.com/posts/jpstauber_big-news-in-typography-microsoft-words-activity-7168729289373200385-UwBe) and call it an event of the same significance as when Google started using Roboto on Android. Try choosing Aptos Light for headings. Aptos [is somewhat wider](https://office-watch.com/2023/aptos-calibri-comparison/) than Calibri, so you don't have to worry about reaching for the narrowed variant Aptos Narrow in narrow text columns.

**Taťána le Moigne is ending her role as head of Google's Czech branch after 18 years.** [Seriously, eighteen years.](https://www.mediar.cz/tatana-le-moigne-po-18-letech-konci-v-cele-ceskeho-googlu/) That's almost half of our lives. *Thank you!* *Good work!*

# Life with Technologies

Websites are built from blocks today, [we wrote in a book](https://reknisioweb.substack.com/p/kniha). And if we can build them from blocks, why couldn't artificial intelligence assemble them for us from those building blocks?

![Laughing girl: When I realize Chat GPT will take my job. Grimacing girl: When I realize...](https://substack-post-media.s3.amazonaws.com/public/images/0fb2418e-ff11-4c5d-939d-08da8d1ec102.avif "320x320")

Twenty years ago, programmers were amazed by scaffolding using an [ER model](https://en.wikipedia.org/wiki/Entity%E2%80%93relationship_model#:~:text=An%20entity%E2%80%93relationship%20model%20(or,instances%20of%20those%20entity%20types) description. The Ruby on Rails framework only required typing in the terminal: A class has one teacher. A teacher has one or more students. And moments later you had a running application with an interface for [CRUD operations](https://en.wikipedia.org/wiki/Create,_read,_update_and_delete#:~:text=In%20computer%20programming%2C%20create%2C%20read,computer%2Dbased%20forms%20and%20reports.).

We've moved forward a bit, so you no longer need to open a terminal and speak the obscure language of database specialists. Tools like [Draw a UI](https://www.draw-a-ui.com/) or [Make Real](https://makereal.tldraw.com/) allow you to doodle a rough interface sketch and generate source code for it. You can [copy it from GitHub](https://www.google.com/search?q=site%3Agithub.com+(make-real+OR+draw-a-ui) and run it on your machine.

But you still need to be able to sketch a bit and think about interface elements. However, directly in the website builder category, you'll find tools that don't even require a sketch. For example, [Hostinger](https://www.hostinger.com/website-builder), [Teleport](https://teleporthq.io/ai-website-builder), [10Web](https://10web.io/ai-website-builder/), [Durable](https://durable.co/ai-website-builder), or the well-known [WiX](https://www.wix.com/ai-website-builder). [Framer](https://www.framer.com/) goes about it a bit differently, but it's certainly a big step forward.

However, if you already have your own [design system](https://www.youtube.com/watch?v=1SnB9SWtibU), you probably want the building block from which the website is built to be precisely that. This is exactly what [Magic Patterns](https://www.magicpatterns.com/ai) offers. If we were to paraphrase Jakob Nielsen: You no longer need to learn to program; AI will solve everything.
