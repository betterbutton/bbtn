# Safari in iOS 18 rids the web of annoying clutter
> 

![Dialog in the Safari browser explains the new feature for removing distracting content.](https://substack-post-media.s3.amazonaws.com/public/images/5124bc41-b9a1-4d90-a394-668b1a9b8b99_1741x918.png "1456x768")

Mac Rummors introduces [the removal of distracting content in Safari under iOS 18](https://www.youtube.com/watch?v=FzBa6Tc-JeE)

The iOS 18 and MacOS Sequoia updates will give Safari users [the option to block pop-ups and other distracting elements on web pages](https://nypost.com/2024/08/06/tech/apple-ios-18-new-distraction-control-feature-explained/) using the new *Distraction Control* feature. This feature, available in beta, will allow users to remove annoying panels and ads. Once enabled, it activates by clicking the page icon next to the URL bar and selecting *Hide Distracting Items*.

> A few years ago I wouldn't have expected Apple of all companies to be saving the Web. But it's awesome and I'm really rooting for it.\
> **— [Martin Michálek](https://www.linkedin.com/feed/update/urn:li:activity:7226810097883099138/)**

The setting is not permanent and must be enabled for each new page visit. The feature does not block all ads permanently and must be activated separately on each device.
